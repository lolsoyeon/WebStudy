■■■ Emmet(에밋 구:ZEN Coding)■■■

○ 개요

	- 기존에 zen coding 이러는 이름으로 사용되었다.
	- 이후에 Google 에서 이 도구를 수용하면서 Emmet 이라는 이름으로 변경되었다.
	- HTML 생산성 도구(HTML 코딩을 빠르게 하기 위한 플러그인)
	- CSS 선택자 사용(CSS3 지원)
	  → HTML 을 CSS 선택자를 통해 검색하는 방법 대신
	 	 CSS 선택자를 통해 HTML 을 생성하도록 처리하는 개념
	
	
○ 환경 설정

	- 대부분의 툴(tool)에서 지원하고 있음
	  (※ 대부분의 편집기 내에서의 액션 키 : Ctrl + E)
	- http://emmet.io
	해당 페이지에서 사용하는 툴에 따른 플러그인 설치
